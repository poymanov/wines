# Новое русское вино

Сайт магазина авторского вина "Новое русское вино".

## Установка

Для работы приложения требуется **Docker** и **Docker Compose**.

## Запуск

Выполнить команду:

```
make start
```

Сайт будет доступен по адресу [http://localhost:8000](http://localhost:8000).

Для остановки сайта:

```
make stop
```
